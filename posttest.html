Posttest: FY BTech Engineering Mathematics

15 advanced MCQs on linear algebra concepts with stepwise solutions

Instructions

  • Use this posttest after reading theory and using simulation.
  • Each question has one correct option and a brief stepwise solution.
  • Topics follow first-year engineering mathematics outcomes.

Posttest Questions (15)

Q1. For vectors in ℝ³, if rank = 3, then the set of exactly 3 vectors is:

Options: A) Dependent B) Independent C) Orthogonal only D) Not defined

Correct Option: B

Stepwise Solution:

  1. Rank = number of independent vectors.
  2. Given 3 vectors and rank 3.
  3. All 3 are independent.

Q2. If matrix columns are vectors in ℝ⁴ and one row in RREF is all zeros, then:

Options: A) Rank is 4 B) Rank is less than 4 C) Determinant is 1 D) Vectors must be orthogonal

Correct Option: B

Stepwise Solution:

  1. Zero row in RREF indicates not full row rank.
  2. So rank < number of rows (4).
  3. Hence rank is less than 4.

Q3. Which is always true for a homogeneous system Ax = 0?

Options: A) No solution B) Exactly one non-zero solution C) Trivial solution always exists D) Infinite solutions always

Correct Option: C

Stepwise Solution:

  1. Set x = 0 vector.
  2. Then A(0)=0 always.
  3. So trivial solution always exists.

Q4. For a 3×3 matrix, non-zero determinant implies:

Options: A) Rank 2 B) Rank 1 C) Rank 3 D) Rank 0

Correct Option: C

Stepwise Solution:

  1. det(A) ≠ 0 means matrix is invertible.
  2. Invertible square matrix has full rank.
  3. Hence rank is 3.

Q5. In ℝⁿ, any set with more than n vectors is:

Options: A) Independent B) Dependent C) Basis D) Orthonormal

Correct Option: B

Stepwise Solution:

  1. Dimension is n.
  2. At most n vectors can be independent.
  3. So more than n vectors must be dependent.

Q6. If vectors v₁,v₂,v₃ satisfy v₃ = 2v₁ - v₂, then {v₁,v₂,v₃} is:

Options: A) Independent B) Dependent C) Orthogonal D) Basis of ℝ³ always

Correct Option: B

Stepwise Solution:

  1. One vector is linear combination of others.
  2. So non-trivial relation exists: 2v₁ - v₂ - v₃ = 0.
  3. Hence set is dependent.

Q7. The rank of identity matrix I₅ is:

Options: A) 0 B) 1 C) 4 D) 5

Correct Option: D

Stepwise Solution:

  1. I₅ has 5 pivot positions.
  2. All rows and columns are independent.
  3. Therefore rank(I₅)=5.

Q8. If 2 vectors in ℝ² are dependent, then geometrically they:

Options: A) Are perpendicular B) Lie on same line through origin C) Form a basis D) Have equal magnitude only

Correct Option: B

Stepwise Solution:

  1. Dependent means one is scalar multiple of other.
  2. Scalar multiples share same direction line.
  3. Thus they lie on same line through origin.

Q9. Which set can form a basis for ℝ²?

Options: A) (1,2),(2,4) B) (0,0),(1,0) C) (1,0),(0,1) D) (3,6),(1,2)

Correct Option: C

Stepwise Solution:

  1. Basis in ℝ² needs 2 independent vectors.
  2. Option C vectors are standard basis and independent.
  3. Hence C forms a basis.

Q10. For matrix A (m×n), rank(A) equals:

Options: A) Number of zero rows always B) Number of pivots in RREF C) n always D) m always

Correct Option: B

Stepwise Solution:

  1. Reduce A to RREF.
  2. Identify pivot positions.
  3. Count pivots; this count is rank.

Q11. If A is 4×4 and rank(A)=3, then det(A) is:

Options: A) 1 B) -1 C) 0 D) Cannot be 0

Correct Option: C

Stepwise Solution:

  1. For 4×4, full rank needed for non-zero determinant.
  2. Given rank 3 < 4, A is singular.
  3. So determinant must be 0.

Q12. If vectors are mutually orthogonal and non-zero in ℝ³, then they are:

Options: A) Dependent B) Independent C) Equal D) Always 4 in number

Correct Option: B

Stepwise Solution:

  1. Non-zero orthogonal vectors cannot be scalar multiples.
  2. Orthogonal set of non-zero vectors is independent.
  3. Hence they are linearly independent.

Q13. In ℝ³, set {(1,0,0),(0,1,0),(1,1,0)} is:

Options: A) Independent B) Dependent C) Basis of ℝ³ D) Orthonormal

Correct Option: B

Stepwise Solution:

  1. Third vector = first + second.
  2. Hence one vector depends on others.
  3. So set is dependent.

Q14. Number of pivot columns in a matrix equals:

Options: A) Determinant B) Rank C) Trace D) Eigenvalue count

Correct Option: B

Stepwise Solution:

  1. Pivot columns represent independent columns.
  2. Count of independent columns is rank.
  3. Therefore pivot count equals rank.

Q15. If A is 2×2 with columns (a,b) and (c,d), independence requires:

Options: A) a+b+c+d=0 B) ad-bc ≠ 0 C) ad+bc=0 D) a=d only

Correct Option: B

Stepwise Solution:

  1. Compute determinant of [[a,c],[b,d]].
  2. det = ad - bc.
  3. Columns are independent iff ad - bc ≠ 0.
