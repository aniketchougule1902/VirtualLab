Get Involved

How to Contribute

We welcome contributions from educators, developers, and students:

For Educators

  • Suggest new example sets or exercises
  • Provide feedback on educational effectiveness
  • Share how you're using the tool in your teaching

For Developers

  • Contribute code via GitHub
  • Report bugs or suggest features
  • Improve documentation

For Students

  • Test the tool and report issues
  • Suggest improvements to user interface
  • Share ideas for new features

GitHub Repository: https://github.com/VIrtualLabs/vector-checker

Project Timeline

Development History

  • January 2023: Project conception and initial planning
  • March 2023: Core algorithm development and testing
  • May 2023: UI/UX design and prototype creation
  • July 2023: Alpha testing with mathematics educators
  • September 2023: Beta release and student testing
  • November 2023: Public release v1.0
  • Future: Planned features include ℝ⁴ visualization, step-by-step solutions, and mobile apps
