Pretest Questions (15)

Q1. Which condition confirms that two vectors in ℝ² are linearly independent?

Options: A) One is zero vector B) One is scalar multiple of other C) Determinant of 2×2 matrix formed by vectors is non-zero D) Their magnitudes are equal

Correct Option: C

Stepwise Solution:

  1. Place vectors as columns in a 2×2 matrix.
  2. Compute determinant.
  3. If determinant ≠ 0, columns are independent.

Q2. The vectors (1,2,3) and (2,4,6) in ℝ³ are:

Options: A) Independent B) Orthogonal C) Dependent D) Unit vectors

Correct Option: C

Stepwise Solution:

  1. Check scalar multiple: (2,4,6)=2(1,2,3).
  2. So one vector is multiple of the other.
  3. Hence vectors are linearly dependent.

Q3. Maximum number of linearly independent vectors in ℝ⁴ is:

Options: A) 2 B) 3 C) 4 D) 5

Correct Option: C

Stepwise Solution:

  1. Dimension theorem: max independent vectors = dimension.
  2. Dimension of ℝ⁴ is 4.
  3. So maximum independent vectors = 4.

Q4. If rank of matrix formed by 3 vectors in ℝ³ is 2, the vectors are:

Options: A) Independent B) Dependent C) Orthogonal D) Equal

Correct Option: B

Stepwise Solution:

  1. Number of vectors = 3.
  2. Rank gives number of independent vectors = 2.
  3. Since rank < number of vectors, set is dependent.

Q5. Determinant test is directly applicable when:

Options: A) Matrix is rectangular B) Number of vectors < dimension C) Matrix is square D) Entries are all integers

Correct Option: C

Stepwise Solution:

  1. Determinant is defined only for square matrices.
  2. Square means rows = columns.
  3. Hence determinant test needs square matrix.

Q6. If det(A)=0 for a 3×3 matrix A made of three vectors in ℝ³, then vectors are:

Options: A) Independent B) Dependent C) Unit D) Parallel only

Correct Option: B

Stepwise Solution:

  1. For square matrix, det(A)=0 implies singular matrix.
  2. Singular matrix has rank < 3.
  3. So column/row vectors are linearly dependent.

Q7. In ℝ², any set of 3 vectors is always:

Options: A) Independent B) Dependent C) Orthogonal D) Basis

Correct Option: B

Stepwise Solution:

  1. Dimension of ℝ² is 2.
  2. Any set with more than 2 vectors cannot all be independent.
  3. Therefore any 3 vectors in ℝ² are dependent.

Q8. RREF of a matrix helps identify:

Options: A) Only determinant B) Only inverse C) Pivot positions and rank D) Only eigenvalues

Correct Option: C

Stepwise Solution:

  1. Convert matrix to RREF.
  2. Count pivot columns.
  3. Pivot count equals rank and indicates independence structure.

Q9. Which set in ℝ³ is linearly independent?

Options: A) (1,0,0),(2,0,0),(0,1,0) B) (1,0,0),(0,1,0),(0,0,1) C) (1,1,1),(2,2,2),(3,3,3) D) (1,2,3),(2,4,6),(3,6,9)

Correct Option: B

Stepwise Solution:

  1. Option B is standard basis vectors.
  2. No vector in B is combination of the other two.
  3. Hence option B is independent.

Q10. If a set contains zero vector, the set is:

Options: A) Always independent B) Always dependent C) Independent in odd dimensions D) Depends on determinant only

Correct Option: B

Stepwise Solution:

  1. Take coefficient 1 for zero vector and 0 for others.
  2. This gives non-trivial combination equal to zero vector.
  3. So set is linearly dependent.

Q11. For vectors as columns of A, if rank(A)=k (number of columns), then columns are:

Options: A) Dependent B) Independent C) Equal D) Orthogonal

Correct Option: B

Stepwise Solution:

  1. Rank equals number of independent columns.
  2. If rank equals total columns k, all are independent.
  3. Hence column vectors are linearly independent.

Q12. Which statement is true for vectors in ℝ⁵?

Options: A) Any 5 vectors are always dependent B) Any 6 vectors are always dependent C) Any 4 vectors are always independent D) Rank can exceed 5

Correct Option: B

Stepwise Solution:

  1. Maximum independent vectors in ℝ⁵ is 5.
  2. So any set with more than 5 vectors is dependent.
  3. Therefore any 6 vectors are always dependent.

Q13. The vectors (1,2) and (-2,1) are:

Options: A) Dependent B) Independent C) Equal D) Zero vectors

Correct Option: B

Stepwise Solution:

  1. Form matrix [[1,-2],[2,1]].
  2. Determinant = 1×1 - (-2×2) = 5 ≠ 0.
  3. Hence vectors are linearly independent.

Q14. If two vectors in ℝ³ are collinear, they are:

Options: A) Independent B) Dependent C) Orthogonal D) Basis of ℝ³

Correct Option: B

Stepwise Solution:

  1. Collinear means one vector is scalar multiple of other.
  2. Scalar-multiple relation gives dependency.
  3. So vectors are linearly dependent.

Q15. A basis of ℝ³ must contain:

Options: A) Exactly 2 vectors B) At most 2 vectors C) Exactly 3 independent vectors D) Any 3 vectors

Correct Option: C

Stepwise Solution:

  1. Basis must be linearly independent and span the space.
  2. Dimension of ℝ³ is 3.
  3. Hence basis needs exactly 3 independent vectors.
