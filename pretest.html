15 MCQs on vectors, matrices, determinants, and basic linear algebra with one final submission

Pretest Questions (15)

Q1. What is the essential condition to add two matrices A and B?

Options: A) They must be square matrices B) They must have the same order (dimensions) C) Number of columns of A must equal number of rows of B D) They must be identity matrices

Correct Option: B

Stepwise Solution:

  1. Matrix addition is done element by element.
  2. Therefore, corresponding elements must exist.
  3. Hence, both matrices must have the exact same number of rows and columns.

Q2. When a matrix is multiplied by a scalar k, which elements are multiplied by k?

Options: A) Only the diagonal elements B) Only the first row C) Only the first column D) Every element in the matrix

Correct Option: D

Stepwise Solution:

  1. Scalar multiplication distributes over the matrix.
  2. Each entry a_ij becomes k * a_ij.
  3. Therefore, all elements are multiplied by k.

Q3. For matrix multiplication AB to be defined, what must be true?

Options: A) A and B must have the same order B) Columns of A = Rows of B C) Rows of A = Columns of B D) A and B must be square matrices

Correct Option: B

Stepwise Solution:

  1. In multiplication, a row of A is multiplied by a column of B.
  2. This requires the length of A's rows to match the length of B's columns.
  3. Hence, number of columns in A must equal number of rows in B.

Q4. The transpose of a matrix is obtained by:

Options: A) Changing signs of all elements B) Interchanging rows and columns C) Reversing the order of rows D) Finding the reciprocal of all elements

Correct Option: B

Stepwise Solution:

  1. The transpose operation flips a matrix over its diagonal.
  2. Row index i and column index j are swapped.
  3. Thus, rows become columns and columns become rows.

Q5. What is the determinant of a 2x2 matrix [[a, b], [c, d]]?

Options: A) ab - cd B) ad + bc C) ad - bc D) ac - bd

Correct Option: C

Stepwise Solution:

  1. Multiply the elements of the main diagonal: a * d.
  2. Multiply the elements of the anti-diagonal: b * c.
  3. Subtract the second product from the first: ad - bc.

Q6. A square matrix A is called a singular matrix if its determinant is:

Options: A) 1 B) 0 C) Negative D) Non-zero

Correct Option: B

Stepwise Solution:

  1. A singular matrix does not have an inverse.
  2. The inverse formula involves dividing by the determinant.
  3. Division by zero is undefined, so a matrix is singular if its determinant is 0.

Q7. What is the rank of the matrix [[1, 2], [2, 4]]?

Options: A) 0 B) 1 C) 2 D) Undefined

Correct Option: B

Stepwise Solution:

  1. The second row [2, 4] is exactly 2 times the first row [1, 2].
  2. Applying row operation R2 = R2 - 2*R1 gives [[1, 2], [0, 0]].
  3. There is only 1 non-zero row left, so the rank is 1.

Q8. Reduce the matrix A = [[1, 1], [1, -1]] to Row Echelon Form. What is the value of the bottom right element?

Options: A) -1 B) -2 C) 0 D) 2

Correct Option: B

Stepwise Solution:

  1. Original matrix: [[1, 1], [1, -1]].
  2. Apply row operation R2 -> R2 - R1 to eliminate the 1 in the second row, first column.
  3. New R2 = [1 - 1, -1 - 1] = [0, -2].
  4. The matrix is now [[1, 1], [0, -2]]. The bottom right element is -2.

Q9. For the system x + y = 3 and 2x - y = 0, what is the augmented matrix [A|B]?

Options: A) [[1, 1, 3], [2, -1, 0]] B) [[1, 1, 0], [2, -1, 3]] C) [[1, 2, 3], [1, -1, 0]] D) [[1, 1], [2, -1]]

Correct Option: A

Stepwise Solution:

  1. The coefficients of x and y form the matrix A: [[1, 1], [2, -1]].
  2. The constants on the right side form the column matrix B: [[3], [0]].
  3. Combine them to form [A|B] = [[1, 1, 3], [2, -1, 0]].

Q10. What is the rank of a 3x3 matrix where all elements are 1?

Options: A) 0 B) 1 C) 2 D) 3

Correct Option: B

Stepwise Solution:

  1. The matrix has all rows identical: R1 = [1, 1, 1], R2 = [1, 1, 1], R3 = [1, 1, 1].
  2. Applying R2 -> R2 - R1 and R3 -> R3 - R1 results in two rows of zeros.
  3. The row echelon form is [[1, 1, 1], [0, 0, 0], [0, 0, 0]].
  4. There is 1 non-zero row, so the rank is 1.

Q11. The dot product of two non-zero perpendicular (orthogonal) vectors is:

Options: A) 1 B) -1 C) 0 D) Product of their magnitudes

Correct Option: C

Stepwise Solution:

  1. Dot product formula is A.B = |A||B|cos(θ).
  2. For perpendicular vectors, the angle θ is 90 degrees.
  3. Since cos(90°) = 0, the dot product is 0.

Q12. What happens when you multiply a matrix by an identity matrix of appropriate size?

Options: A) It becomes a null matrix B) It becomes an identity matrix C) The original matrix remains unchanged D) Its transpose is obtained

Correct Option: C

Stepwise Solution:

  1. The identity matrix I acts like the number 1 for matrix multiplication.
  2. Therefore, AI = A and IA = A.
  3. The original matrix is unchanged.

Q13. In the augmented matrix [[1, 2, | 5], [2, 3, | 8]], applying the operation R2 -> R2 - 2*R1 yields which second row?

Options: A) [0, -1, | -2] B) [0, 1, | 2] C) [0, 0, | 0] D) [1, -1, | -3]

Correct Option: A

Stepwise Solution:

  1. Initial R2 is [2, 3, 8] and R1 is [1, 2, 5].
  2. Multiply R1 by 2: [2, 4, 10].
  3. Subtract from R2: [2-2, 3-4, 8-10] = [0, -1, -2].
  4. The new second row is [0, -1, | -2].

Q14. What is the maximum possible rank of a matrix of order 3x4?

Options: A) 4 B) 3 C) 7 D) 12

Correct Option: B

Stepwise Solution:

  1. The rank of an m x n matrix cannot exceed the smaller of m and n.
  2. Here, the dimensions are 3 and 4.
  3. The smaller value is 3, so the maximum possible rank is 3.

Q15. The system of equations AX = B has a unique solution if the determinant of coefficient matrix A is:

Options: A) Zero B) Non-zero C) Equal to 1 D) Equal to rank of B

Correct Option: B

Stepwise Solution:

  1. For a unique solution, the matrix A must be invertible.
  2. A matrix is invertible only if its determinant is not equal to zero.
  3. Hence, det(A) must be non-zero for a unique solution.
