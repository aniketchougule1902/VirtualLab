Help & Instructions

How to Use This Tool

  1. Select 2D or 3D dimension for your vectors
  2. Add vectors by clicking "Add Vector" button
  3. Enter vector components in the input fields
  4. Click "Analyze Vectors" to check linear dependency
  5. View mathematical results and graphical representation
  6. Use export buttons to save your results

Understanding Linear Dependency

Linearly Independent: No vector in the set can be written as a linear combination of the others.

Linearly Dependent: At least one vector in the set can be written as a linear combination of the others.

In 2D: Two vectors are dependent if they lie on the same line (collinear).

In 3D: Three vectors are dependent if they lie on the same plane (coplanar).

Mathematical Methods Used

  • Rank: Maximum number of linearly independent vectors
  • Determinant: For square matrices, zero determinant indicates dependency
  • RREF: Reduced Row Echelon Form to identify pivot columns

About MathVortexLabs

MathVortexLabs is a static web application for analyzing linear dependency/independency of vectors in 2D and 3D space using both mathematical and graphical approaches.

This tool is designed to help students and educators visualize and understand linear algebra concepts without requiring any external libraries or dependencies.

Features:

  • 2D & 3D Vector Analysis
  • Mathematical Approach (Rank, Determinant, RREF)
  • Graphical Approach (Visual Representation)
  • 3D Plotting with interactive rotation
  • Responsive Design
  • Export Results & Plots

Technologies: Pure HTML5, CSS3, JavaScript

Created for educational purposes. All calculations are performed client-side.
