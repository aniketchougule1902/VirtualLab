Linear Algebra Theory

Easy explanations of vectors, span, rank, basis, and linear dependence

The Big Idea

Linear algebra studies how vectors work together. In this lab, the main question is simple: can one vector be built from the others, or not?

If a vector can be made from the others, the set is dependent. If every vector adds something new, the set is independent.

Quick memory tip: independent vectors bring new directions; dependent vectors repeat information.

What Is a Vector?

Meaning

A vector is an ordered list of numbers such as (2, 3) or (1, -1, 4). Each number is called a component.

Dimension

The number of components tells us the dimension. For example, (2, 3) is in ℝ² and (1, -1, 4) is in ℝ³.

Interpretation

In geometry, a vector can be seen as an arrow. In algebra, it is a list of values that can be scaled and added.

Linear Combination and Span

Linear Combination

A linear combination is made by multiplying vectors by numbers and then adding the results.

a₁v₁ + a₂v₂ + ... + aₖvₖ

Span

The span of a set of vectors is the collection of all vectors you can create from those linear combinations.

  • One non-zero vector in ℝ² spans a line through the origin.
  • Two independent vectors in ℝ² span the whole plane.
  • Three independent vectors in ℝ³ span the whole 3D space.

Linear Dependence and Independence

Dependent Set

A set is dependent when at least one vector can be written as a combination of the others.

That means the set contains repeated direction or repeated information.

Independent Set

A set is independent when no vector in the set can be made from the others.

Each vector is necessary and contributes a new direction.

Simple Example

The vectors (1, 2, 3) and (2, 4, 6) are dependent because the second vector is just 2 times the first.

The vectors (1, 0, 0), (0, 1, 0), and (0, 0, 1) are independent because none of them can be formed from the other two.

Introduction to Vectors in ℝⁿ (Very Simple)

Definition: n-Dimensional Vector

A vector in ℝⁿ is just an ordered list of n real numbers.

v = (v₁, v₂, ..., vₙ) ∈ ℝⁿ

Each value vᵢ is called a component.

Linear Combination in n-Dimensions

Multiply each vector by a number (scalar), then add.

w = c₁v₁ + c₂v₂ + ... + cₖvₖ

Dependency & Independency (Main Condition)

Check this equation:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

  • Linearly dependent: if there is a solution where at least one c-value is non-zero.
  • Linearly independent: if the only solution is c₁ = c₂ = ... = cₖ = 0.

ℝ² (2D)

  • Dependent if vectors lie on the same line (collinear).
  • Any 3 or more vectors are always dependent.

ℝ³ (3D)

  • Dependent if vectors lie in the same plane (coplanar).
  • Any 4 or more vectors are always dependent.

ℝⁿ (nD)

  • General rule: more than n vectors are always dependent.
  • Maximum independent vectors in ℝⁿ is exactly n.
1

Matrix Rank Method

Form a matrix using vectors as columns. If rank = number of vectors, independent; otherwise dependent.

2

Determinant Method

This method works only in the square case (n vectors in ℝⁿ): det(A) ≠ 0 means independent, det(A) = 0 means dependent.

3

RREF Method

Count pivots (leading 1s). Pivots = rank. If rank equals number of vectors, independent; else dependent.

Rank, Determinant, and RREF

1

Rank Test

Rank tells us how many vectors in the set are actually independent. If rank equals the number of vectors, the set is independent.

2

Determinant Test

This works only for square matrices. A non-zero determinant means the vectors are independent. A zero determinant means they are dependent.

3

RREF Test

RREF shows pivot positions clearly. The number of pivots gives the rank, and pivot columns help identify basis vectors.

Basis and Dimension

Basis

A basis is a set of vectors that is both independent and able to span the space.

So, a basis has exactly the right number of vectors: not too few and not too many.

In ℝⁿ, you can never have more than n independent vectors. So any set with more than n vectors is automatically dependent.

Geometric Understanding

In 2D

Two independent vectors point in different directions and can cover the whole plane. If they lie on the same line, they are dependent.

In 3D

Three independent vectors can fill 3D space. If one lies in the plane formed by the other two, the set becomes dependent.

Above 3D

We cannot draw higher-dimensional spaces directly, but the same algebraic rules still work. That is why rank and RREF are very useful.
