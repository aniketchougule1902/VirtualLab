Linear Algebra Theory

Understanding linear dependency, independency, and related concepts

Introduction to Vectors

In linear algebra, a vector is an element of a vector space. For the purposes of this tool, we consider vectors in ℝ² (2D) and ℝ³ (3D) spaces. A vector is typically represented as an ordered list of numbers called components or coordinates.

Definition: Vector

A vector in ℝⁿ is an ordered n-tuple of real numbers, typically written as:

v = (v₁, v₂, ..., vₙ)

where each vᵢ is a real number called a component of the vector.

Linear Combination

A linear combination of vectors is an expression constructed from a set of vectors by multiplying each vector by a scalar and adding the results.

Definition: Linear Combination

Given vectors v₁, v₂, ..., vₖ in ℝⁿ and scalars c₁, c₂, ..., cₖ in ℝ, the vector

w = c₁v₁ + c₂v₂ + ... + cₖvₖ

is called a linear combination of v₁, v₂, ..., vₖ.

Example

Given vectors v₁ = (1, 2) and v₂ = (3, 4) in ℝ², and scalars c₁ = 2, c₂ = -1:

w = 2(1, 2) + (-1)(3, 4) = (2, 4) + (-3, -4) = (-1, 0)

So w = (-1, 0) is a linear combination of v₁ and v₂.

Linear Dependency and Independency

Definition: Linearly Dependent Vectors

A set of vectors {v₁, v₂, ..., vₖ} in ℝⁿ is linearly dependent if there exist scalars c₁, c₂, ..., cₖ, not all zero, such that:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

where 0 is the zero vector. If the only solution is c₁ = c₂ = ... = cₖ = 0, the vectors are linearly independent.

Definition: Linearly Independent Vectors

A set of vectors {v₁, v₂, ..., vₖ} in ℝⁿ is linearly independent if the only scalars c₁, c₂, ..., cₖ that satisfy:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

are c₁ = c₂ = ... = cₖ = 0.

Geometric Interpretation

In ℝ² (2D Space)

Two vectors are linearly dependent if and only if they lie on the same line (are collinear).

Three or more vectors in ℝ² are always linearly dependent because the dimension of ℝ² is 2.

In ℝ³ (3D Space)

Two vectors are linearly dependent if they lie on the same line.

Three vectors are linearly dependent if they lie on the same plane (are coplanar).

Four or more vectors in ℝ³ are always linearly dependent because the dimension of ℝ³ is 3.

Important Note

The maximum number of linearly independent vectors in ℝⁿ is n. This number is called the dimension of the vector space.

Methods for Determining Linear Dependency

1. Rank of a Matrix

The rank of a matrix is the maximum number of linearly independent row vectors (or column vectors) in the matrix. For a set of vectors, we form a matrix with the vectors as rows (or columns) and compute its rank.

Rank and Linear Independence

Given k vectors in ℝⁿ, form an n × k matrix (or k × n matrix). The vectors are linearly independent if and only if the rank of this matrix equals k (if k ≤ n).

2. Determinant Method (for Square Matrices)

For n vectors in ℝⁿ, we can form an n × n matrix. The vectors are linearly independent if and only if the determinant of this matrix is non-zero.

Example in ℝ²

Given vectors v₁ = (a, b) and v₂ = (c, d), form the matrix:

A = \begin{bmatrix} a & b \\ c & d \end{bmatrix}

The vectors are linearly independent if and only if:

det(A) = ad - bc ≠ 0

Example in ℝ³

Given vectors v₁ = (a₁, a₂, a₃), v₂ = (b₁, b₂, b₃), and v₃ = (c₁, c₂, c₃), form the matrix:

A = \begin{bmatrix} a₁ & a₂ & a₃ \\ b₁ & b₂ & b₃ \\ c₁ & c₂ & c₃ \end{bmatrix}

The vectors are linearly independent if and only if det(A) ≠ 0.

3. Reduced Row Echelon Form (RREF)

The RREF of a matrix reveals its rank and pivot positions. The vectors corresponding to pivot columns are linearly independent.

Using RREF

After reducing the matrix to RREF:

  • Count the number of pivot columns (columns with leading 1's)
  • This number equals the rank of the matrix
  • If rank = number of vectors, they are linearly independent
  • If rank < number of vectors, they are linearly dependent

Applications and Importance

Why Linear Independence Matters

  • Basis for Vector Spaces: Linearly independent sets that span a vector space form a basis.
  • Solving Linear Equations: Linear independence determines whether a system has a unique solution.
  • Dimension Theory: The maximum number of linearly independent vectors defines the dimension of a space.
  • Coordinate Systems: Basis vectors provide coordinate systems for vector spaces.
  • Data Analysis: Principal Component Analysis (PCA) uses linear independence concepts.

Key Theorem: Dimension Theorem

If a vector space V has a basis with n vectors, then every basis of V has exactly n vectors. This number n is called the dimension of V, denoted dim(V).

Practice Exercises

Exercise 1

Determine if the vectors v₁ = (1, 2, 3), v₂ = (4, 5, 6), and v₃ = (7, 8, 9) are linearly independent.

Hint: Check if the determinant of the matrix formed by these vectors is zero.

Exercise 2

Find the value of k for which the vectors v₁ = (1, 2) and v₂ = (3, k) are linearly dependent.

Hint: Two vectors in ℝ² are linearly dependent if one is a scalar multiple of the other.

Exercise 3

Show that any set containing the zero vector is linearly dependent.

Hint: Consider the linear combination with coefficient 1 for the zero vector and 0 for all others.
