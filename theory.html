Linear Algebra Theory

Understanding linear dependency, independency in n-dimensional spaces

Vectors in ℝⁿ

A vector in ℝⁿ is an ordered group of real numbers. The number of components is the dimension, and each component describes one coordinate of the vector.

A set of vectors can be studied algebraically in any dimension, even when it cannot be drawn geometrically.

Linear Combination and Span

A linear combination is formed by multiplying vectors by scalars and adding them. The set of all such combinations is called the span.

Span shows which vectors can be generated from a given set and helps determine whether that set is sufficient to describe a space or subspace.

Linear Dependence and Independence

A set is linearly dependent if at least one vector can be written as a combination of the others. A set is linearly independent when this is not possible.

In ℝⁿ, any set containing more than n vectors is always dependent. This is a direct result of dimension.

Main Tests Used in This Lab

Rank Test

Rank gives the maximum number of independent vectors in the set. If rank equals the number of vectors, the set is independent.

Determinant Test

For square matrices only, a non-zero determinant indicates independence, while zero indicates dependence.

RREF Test

RREF reveals pivot positions. The number of pivots equals rank and helps identify basis vectors directly.

Dimensions Above 3

For dimensions greater than 3, visualization is limited, but the same linear algebra rules still apply. The lab therefore provides full mathematical output for higher dimensions and graph output for 2D and 3D.
