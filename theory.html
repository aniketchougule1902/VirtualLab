Linear Algebra Theory

Understanding linear dependency, independency in n-dimensional spaces

Introduction to Vectors in ℝⁿ

In linear algebra, a vector is an element of a vector space. For n-dimensional space ℝⁿ, a vector is represented as an ordered n-tuple of real numbers.

Definition: n-Dimensional Vector

A vector in ℝⁿ is an ordered n-tuple of real numbers:

v = (v₁, v₂, ..., vₙ) ∈ ℝⁿ

where each vᵢ is a real number called a component of the vector. The number n is called the dimension of the vector.

Linear Combination in n-Dimensions

A linear combination of vectors in ℝⁿ is constructed by multiplying each vector by a scalar and adding the results.

Definition: Linear Combination in ℝⁿ

Given vectors v₁, v₂, ..., vₖ in ℝⁿ and scalars c₁, c₂, ..., cₖ in ℝ, the vector

w = c₁v₁ + c₂v₂ + ... + cₖvₖ

is called a linear combination of v₁, v₂, ..., vₖ. The set of all such linear combinations is called the span of the vectors.

Linear Dependency and Independency in ℝⁿ

Definition: Linearly Dependent Vectors in ℝⁿ

A set of vectors {v₁, v₂, ..., vₖ} in ℝⁿ is linearly dependent if there exist scalars c₁, c₂, ..., cₖ, not all zero, such that:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

where 0 is the zero vector in ℝⁿ. If the only solution is c₁ = c₂ = ... = cₖ = 0, the vectors are linearly independent.

Geometric Interpretation by Dimension

In ℝ² (2D Space)

Two vectors are linearly dependent if they lie on the same line (collinear).

Three or more vectors in ℝ² are always linearly dependent because the dimension of ℝ² is 2.

In ℝ³ (3D Space)

Two vectors are linearly dependent if they lie on the same line.

Three vectors are linearly dependent if they lie on the same plane (coplanar).

Four or more vectors in ℝ³ are always linearly dependent because the dimension of ℝ³ is 3.

In ℝⁿ (n-Dimensional Space)

General Rule: In ℝⁿ, any set of more than n vectors is always linearly dependent.

Maximum independent set: The maximum number of linearly independent vectors in ℝⁿ is n.

Visualization limitation: For n > 3, we cannot visualize vectors geometrically, but the mathematical principles still apply.

Dimension Theorem (Fundamental Theorem of Linear Algebra)

The maximum number of linearly independent vectors in ℝⁿ is n. This number n is called the dimension of the vector space.

For any vector space V, if V has a basis with n vectors, then every basis of V has exactly n vectors.

Methods for Determining Linear Dependency in ℝⁿ

1. Rank of a Matrix

The rank of a matrix is the maximum number of linearly independent row vectors (or column vectors). For k vectors in ℝⁿ, form a matrix with the vectors as rows (or columns).

Rank Test for Linear Independence

Given k vectors in ℝⁿ, form a k × n matrix (vectors as rows) or n × k matrix (vectors as columns). The vectors are linearly independent if and only if the rank of this matrix equals k (provided k ≤ n).

2. Determinant Method

For n vectors in ℝⁿ (square matrix), we can compute the determinant. The vectors are linearly independent if and only if the determinant is non-zero.

Determinant in Higher Dimensions

For vectors v₁, v₂, ..., vₙ in ℝⁿ, form the n × n matrix:

A = \begin{bmatrix} v₁ \\ v₂ \\ \vdots \\ vₙ \end{bmatrix}

The vectors are linearly independent if and only if det(A) ≠ 0.

Note: This method only works when number of vectors equals dimension (square matrix).

3. Reduced Row Echelon Form (RREF)

The RREF reveals the rank and pivot positions. The number of pivot columns equals the rank.

Using RREF for Any Number of Vectors

After reducing the matrix to RREF:

  • Count the number of pivot columns (columns with leading 1's)
  • This number equals the rank of the matrix
  • If rank = number of vectors, they are linearly independent
  • If rank < number of vectors, they are linearly dependent
  • The vectors corresponding to pivot columns form a basis for the span

Working with Higher Dimensions (n > 3)

Mathematical Analysis Without Visualization

For dimensions higher than 3, we cannot visualize vectors geometrically, but all mathematical methods still apply:

  1. Rank calculation works for any dimension
  2. RREF computation works for any dimension
  3. Determinant calculation works for square matrices
  4. Linear dependency tests are purely algebraic

Example: 4D Vectors

Consider vectors in ℝ⁴:

v₁ = (1, 0, 0, 0), v₂ = (0, 1, 0, 0), v₃ = (0, 0, 1, 0), v₄ = (0, 0, 0, 1)

These are linearly independent (standard basis of ℝ⁴). The matrix formed has rank 4, and its determinant is 1.

Now consider:

v₁ = (1, 2, 3, 4), v₂ = (2, 4, 6, 8), v₃ = (3, 6, 9, 12)

These are linearly dependent because v₂ = 2v₁ and v₃ = 3v₁. The rank of the matrix is 1.

Applications of n-Dimensional Vector Analysis

Real-World Applications

  • Data Science: Feature vectors in machine learning often have hundreds or thousands of dimensions
  • Physics: Phase space in classical mechanics can have 6N dimensions for N particles
  • Economics: Portfolio optimization with n assets
  • Computer Graphics: Homogeneous coordinates (4D) for 3D transformations
  • Quantum Mechanics: State vectors in Hilbert spaces (infinite-dimensional)

Practice Exercises

Exercise 1 (4D Space)

Determine if the vectors v₁ = (1, 0, 0, 1), v₂ = (0, 1, 0, 1), v₃ = (0, 0, 1, 1), and v₄ = (1, 1, 1, 3) are linearly independent in ℝ⁴.

Hint: Form a 4×4 matrix and compute its rank or determinant.

Exercise 2 (5D Space)

Find the maximum number of linearly independent vectors possible in ℝ⁵.

Answer: 5 (by the dimension theorem)

Exercise 3 (General n-D)

Prove that in ℝⁿ, any set containing the zero vector is linearly dependent.

Proof: Take the linear combination with coefficient 1 for the zero vector and 0 for all others. This gives a non-trivial linear combination equal to zero.
