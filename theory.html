Linear Algebra Theory

Easy explanations of vectors, span, rank, basis, and linear dependence

The Main Idea

In this lab, we check one thing: can one vector be made using the other vectors?

If yes, the vectors are dependent. If no, the vectors are independent.

What Is a Vector?

What is a vector?

A vector is a list of numbers like (2, 3) or (1, -1, 4).

v = (v₁, v₂, ..., vₙ)

Linear combination

Multiply vectors by numbers, then add them.

w = c₁v₁ + c₂v₂ + ... + cₖvₖ

How to Check Dependence

Basic rule

Check this equation:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0

  • Dependent: the linear combination equals zero when at least one c-value is non-zero.
  • Independent: only c₁ = c₂ = ... = cₖ = 0 makes the linear combination equal zero.

In ℝ²

  • Vectors on the same line are dependent.
  • Any 3 or more vectors are always dependent.

In ℝ³

  • Vectors in one plane can be dependent.
  • Any 4 or more vectors are always dependent.

In ℝⁿ

  • More than n vectors are always dependent.
  • Maximum independent vectors = n.

Methods Used in This Lab

1

Rank

If rank = number of vectors, they are independent. Otherwise dependent.

2

Determinant

For square matrix only: det ≠ 0 means independent, det = 0 means dependent.

3

RREF

Count pivots. Pivot count gives rank and helps decide dependence.
